# MAC 系统常用快捷键
| 功能               | 按键            |
| ------------------ | --------------- |
| 隐藏或显示Dock     | `⌥` + `⌘` + `D` |
| 隐藏或显示隐藏文件 | `⌘`+ `⇧`+ `。`  |
| 聚焦搜索           | `⌘`+`space`     |
| 新建窗口           | `⌘`+`N`         |
| 新建标签页         | `⌘`+`T`         |
| 全屏APP            | `⌃`+`⌘`+`F`     |
| 截屏/录屏          | `⇧`+`⌘`+`5`     |
| 打开APP设置        | `⌘`+`,`         |
| 移动到废纸篓       | `⌘`+`⌫`         |
